# Mini Project 3

## Description

This project will be using Flask to deploy a small web app called VinylVault to track personal vinyl album collection.

## Getting Started

### Dependencies

```
pip install -r requirement.txt
```

### Executing program

```
python main.py
```

### Output

This web-app should allow users to create a personal account, add albums owned, wishlist albums, and track their collection value.
